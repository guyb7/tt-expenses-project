# Expenses Tracking App
A webapp to track expenses in your company.
Backend: Node, Express, PostgreSQL
Frontend: React, Redux

## Installation
node v8.1.4
yarn v0.24.6

git clone
create `config/config.env`
yarn install

### Config file
**config/config.env**
APP_NAME=expenses-app
ENV=development
PORT=3003
